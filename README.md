# 🚀 CareerOS

> AI-Powered Career Operating System

CareerOS is an intelligent career automation platform that helps users discover jobs, analyze resumes, optimize ATS scores, generate cover letters, automate applications, and track their career journey from a single dashboard.

---

# 🌟 Features

## ✅ Backend

- FastAPI REST API
- SQLAlchemy ORM
- SQLite Database
- Repository Pattern
- Service Layer
- Modular Architecture
